# 🎓 Cours Frameworks Python

## À propos
Ce dépôt contient mes **travaux pratiques et exercices réalisés dans le cadre de mes cours à l’école** sur les frameworks Python.  
Il s’agit d’un espace d’apprentissage, pas de projets professionnels finalisés.  

## Frameworks étudiés
- **Flask** : initiation au développement web léger.  
- **Django** : apprentissage d’un framework complet et structuré.  
- **Tornado** : exploration des applications asynchrones.  

## Organisation du dépôt
- `module0_frameworks_python/` → Introduction générale aux frameworks Python  
- `module1_flask/` → TP CRUD Étudiants avec Flask  
- `module2_Tornado/` → Exercices Tornado
 - `module4_django/` → Exercices Django

## Objectif
👉 Montrer que je suis **en formation** et que ce dépôt est un **support pédagogique**.  
Il sert à documenter mon progression et mes expérimentations au fil des modules.

## Remarque
- Les fichiers `.env` et configurations locales (`.idea`, etc.) sont ignorés pour protéger les données sensibles et garder le dépôt propre.  
- Ce dépôt est avant tout un **carnet de cours** et non une vitrine de projets terminés.
